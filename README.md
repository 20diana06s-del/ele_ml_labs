## 🛠 Как выполнять и заполнять отчеты

Отчёты заполняются **непосредственно в Markdown-файлах** (`lab1.md`, `lab2.md`). Вам не нужно создавать отдельные файлы — просто впишите свои ответы и вставьте графики в соответствующие секции.

### Рекомендуемые инструменты для заполнения

1. **[Obsidian](https://obsidian.md/) (Рекомендуется)**
   - Позволяет вставлять скриншоты из буфера обмена (`Ctrl+V`).
   - Корректно отображает все математические формулы.
   - Просто откройте папку с работами как "Vault" (Хранилище).

2. **Visual Studio Code**
   - Установите расширение **Markdown All in One**.
   - Для вставки картинок установите расширение **Paste Image** (`Ctrl+Alt+V`).
   - Используйте `Ctrl+Shift+V` для предпросмотра.

3. **Typora**
   - Удобный редактор с мгновенным отображением результата (WYSIWYG).

### Как вставлять графики

1. Сделайте скриншот графика в Excel (Win+Shift+S).
2. Поставьте курсор в Markdown-файл под заголовком задания.
3. Нажмите `Ctrl+V` (в Obsidian) или используйте плагин в VS Code.

## 🐍 Установка Python и запуск Jupyter Notebooks

Для лабораторных работ с Python (Lab 3+) используется менеджер пакетов **uv** — быстрый и современный аналог pip.

### 1. Установка uv

**Windows (PowerShell):**
```powershell
irm https://astral.sh/uv/install.ps1 | iex
```

**macOS / Linux:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Проверка установки:
```bash
uv --version
```

### 2. Клонирование репозитория

```bash
git clone <url-репозитория>
cd ee_ml_labs
```

### 3. Установка зависимостей

```bash
uv sync
```

Эта команда создаст виртуальное окружение `.venv` и установит все необходимые пакеты:
- numpy, pandas, matplotlib, seaborn
- scikit-learn
- jupyter

### 4. Запуск в VS Code

1. **Установите VS Code:** https://code.visualstudio.com/

2. **Установите расширения:**
   - **Python** (Microsoft)
   - **Jupyter** (Microsoft)
   - **Pylance** (Microsoft)

3. **Откройте репозиторий:**
   ```bash
   code .
   ```

4. **Выберите интерпретатор Python:**
   - Нажмите `Ctrl+Shift+P`
   - Введите "Python: Select Interpreter"
   - Выберите `.venv` из папки проекта (путь должен содержать `ee_ml_labs/.venv`)

5. **Откройте .ipynb файл:**
   - Дважды кликните на файл (например, `lab3-decision-trees.ipynb`)
   - Если попросит выбрать ядро — выберите Python из `.venv`

6. **Запуск ячеек:**
   - `Shift+Enter` — выполнить текущую ячейку и перейти к следующей
   - `Ctrl+Enter` — выполнить текущую ячейку
   - Нажмите "Run All" для выполнения всех ячеек

### 5. Альтернатива: Jupyter в браузере

```bash
uv run jupyter lab
```

Откроется браузер с Jupyter Lab, где можно запускать ноутбуки.

## ⚠️ Частые проблемы

| Проблема | Решение |
|----------|---------|
| "Kernel not found" | Выберите интерпретатор из `.venv` |
| Модуль не найден | Выполните `uv sync` |
| Графики не отображаются | Добавьте `%matplotlib inline` в первую ячейку |
| Ошибки с путями | Убедитесь, что рабочая директория — папка с ноутбуком |
