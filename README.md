## 🛠 Как выполнять и заполнять отчеты

Отчёты заполняются **непосредственно в Markdown-файлах** (`lab1.md`, `lab2.md`). Вам не нужно создавать отдельные файлы — просто впишите свои ответы и вставьте графики в соответствующие секции.

### Рекомендуемые инструменты для заполнения

1. **[Obsidian](https://obsidian.md/) (Рекомендуется)**
   - Позволяет вставлять скриншоты из буфера обмена (`Ctrl+V`).
   - Корректно отображает все математические формулы.
   - Просто откройте папку с работами как "Vault" (Хранилище).

2. **Visual Studio Code**
   - Установите расширение **Markdown All in One**.
   - Для вставки картинок установите расширение **Paste Image** (`Ctrl+Alt+V`).
   - Используйте `Ctrl+Shift+V` для предпросмотра.

3. **Typora**
   - Удобный редактор с мгновенным отображением результата (WYSIWYG).

### Как вставлять графики

1. Сделайте скриншот графика в Excel (Win+Shift+S).
2. Поставьте курсор в Markdown-файл под заголовком задания.
3. Нажмите `Ctrl+V` (в Obsidian) или используйте плагин в VS Code.

## 📊 Математические формулы

В лабах используется синтаксис LaTeX для формул (через KaTeX).

- Уравнение регрессии: $Y = a + bX$
- Метрики: $\text{MAE}$, $\text{MSE}$, $R^2$

Если формула отображается как обычный текст с символами `$`, убедитесь, что ваш редактор поддерживает Markdown с математикой.
